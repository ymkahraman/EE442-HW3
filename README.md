Komutlar

1) Format 

./myfs disk.image -format

2) Write

./myfs disk.image -write source_file dest_file

3) Read

./myfs disk.image -write source_file dest_file

4) List

./myfs disk.image -list

5) Delete

./myfs disk.image -delete file_name

6) PrintFileList

./myfs disk.image -printfilelist

7) PrintFAT

./myfs disk.image -printFAT

8) DeFragmentation

./myfs disk.image -defragment
